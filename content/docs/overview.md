# Start Here

This wiki is focused on the useful part: how to get things, how to use them, and what they actually do in play.

## Quick start

1. Get the ASEOHA manual.
2. Make the Sonic Workbench

1. Use the Spectrometer to unlock:
  - exteriors
  - consoles
  - interiors
2. Use the Quantiscope to make:
  - advanced modular controls
  - Hammer Space Pouch
  - Deadlocker
  - upgrade parts
  - Sonic Laser
3. Build up to late-game TARDIS upgrades:
  - Tesseract Drive
  - Atrium
  - Laser Sonic
  - Tribophysical macro kinetic waveform extrapolator

## How To Get The Manual

Throw Green Dye and a Book into a crafting table
